# Класс Button

## Назначение
Класс кнопки фреймворка LOTIS, наследующий `Element`. Предназначен для создания HTML-кнопок с возможностью назначения обработчиков событий и стилизации. Класс автоматически устанавливает тег `<button>` и атрибут `type="button"`, что отличает его от стандартных `Input`-кнопок. Поддерживает интеграцию с jQuery UI через метод `ui()` для применения стандартных тем оформления. Используется в формах, панелях инструментов и диалоговых окнах для инициирования действий пользователя.

## Свойства

### Наследуемые свойства

| Свойство | Тип | Видимость | Описание |
|----------|-----|-----------|----------|
| **$id** | `string` | `public` | Наследуется от `Construct`. Уникальный идентификатор объекта, используется как HTML-атрибут `id`. |
| **$tagname** | `string` | `public` | Наследуется от `Element`. Устанавливается в `'button'` в конструкторе. Определяет HTML-тег элемента при рендеринге. |
| **$caption** | `string` | `public` | Наследуется от `Element`. Текстовое содержимое кнопки. Отображается как текст внутри тега `<button>`. |
| **$type** | `string` | `public` | Наследуется от `Element`. Устанавливается в `'html'`. Определяет тип объекта для фильтрации и обработки. |
| **$classname** | `string` | `public` | Наследуется от `Element`. CSS-классы элемента, управляются методами `addclass()`, `removeclass()`, `hasclass()`. |
| **$storage** | `array` | `public` | Наследуется от `Quark`. Хранилище содержит секции `'childs'` (потомки), `'element'` (DOM-элемент), `'attr'` (HTML-атрибуты). |

## Методы

### Конструктор

| Метод | Параметры | Возвращает | Описание |
|-------|-----------|------------|----------|
| **__construct** | `string $id = ''` | — | Конструктор класса. Вызывает родительский конструктор `Element`, устанавливает `tagname` в `'button'`, добавляет атрибут `type` со значением `'button'`. |

### Применение стилей

| Метод | Параметры | Возвращает | Описание |
|-------|-----------|------------|----------|
| **ui** | — | `$this` | Добавляет классы jQuery UI для стилизации кнопки. Добавляет классы `'ui-button'`, `'ui-corner-all'`, `'ui-widget'` через `addclass()`. Возвращает `$this` для поддержки fluent interface. |

## Пример использования

```php
$mybutton = LTS::Button()
    ->ui()
    ->capt('Нажать')
    ->attr('disabled', false)
    ->click(
<<<JS
        alert('Hello, world!');
JS
);
```

### Имитация нажатия на клиенте

```JS
LTS(mybutton).click();
```


## Примечания
*   Класс расширяет `Element`, поэтому наследует все методы управления потомками, стилями, атрибутами и событиями.
*   Класс расширяет `Quark` и `Construct`, поэтому поддерживает полный жизненный цикл объектов: `compile()`, `shine()`, `childs()`, `addinspace()`, а также хуки (`$check`, `$before`, `$on` и др.).
*   **jQuery UI:** Классы `ui-button`, `ui-corner-all`, `ui-widget` применяются через `ui()` — для стандартного оформления.
*   **Метод ui():** Требует подключения jQuery UI CSS (`jquery-ui.css`) для корректного отображения стилей.
*   **Текст кнопки:** Устанавливается через свойство `$caption` или метод `capt()` — отображается между тегами `<button>` и `</button>`.
*   **Обработчики событий:** Назначаются через метод `click()` или `on()` — генерируют клиентский JavaScript-код при компиляции.
*   **Идентификатор:** ID кнопки используется для связи с LTS-объектом через `LTS.get('{id}')` или LTS(var), где var - переменная, ссылающаяся на объект класса Button.
*   **Атрибуты:** Дополнительные атрибуты добавляются через `attr()` — `disabled`, `data-*`, `title` и др.
*   **Стили:** CSS-свойства добавляются через `css()` — `width`, `height`, `margin`, `padding` и др.
*   **Потомки:** Кнопка может содержать потомков через `add()` — иконки, изображения и др.
*   **Классы:** CSS-классы добавляются через `addclass()` — для кастомизации внешнего вида.
