# Класс Dialog

## Назначение
Класс модальных диалоговых окон фреймворка LOTIS, наследующий `Div`. Предназначен для создания интерактивных всплывающих окон на основе jQuery UI Dialog. Поддерживает настройку заголовка, эффектов показа/скрытия, модального режима, кнопок действия и позиционирования. Класс автоматически генерирует клиентский JavaScript-код для инициализации диалога и предоставляет методы для программного открытия/закрытия окна. Интегрируется с системой событий LOTIS для обработки нажатий кнопок.

## Свойства

### Наследуемые свойства

| Свойство | Тип | Видимость | Описание |
|----------|-----|-----------|----------|
| **$id** | `string` | `public` | Наследуется от `Construct`. Уникальный идентификатор объекта, используется как HTML-атрибут `id` и для связи с клиентским объектом в `ltsDialog`. |
| **$type** | `string` | `public` | Наследуется от `Element`. Устанавливается в `'html'`. Определяет тип объекта для фильтрации и обработки. |
| **$tagname** | `string` | `public` | Наследуется от `Div`. По умолчанию `'div'`. Определяет HTML-тег элемента при рендеринге. |
| **$classname** | `string` | `public` | Наследуется от `Element`. Автоматически добавляется класс `'Dialog'` в конструкторе. Управляется методами `addclass()`, `removeclass()`, `hasclass()`. |
| **$storage** | `array` | `public` | Наследуется от `Quark`. Хранилище содержит секции `'childs'` (потомки), `'element'` (DOM-элемент), `'options'` (параметры диалога), `'buttons'` (кнопки действия). |

## Методы

### Конструктор

| Метод | Параметры | Возвращает | Описание |
|-------|-----------|------------|----------|
| **__construct** | `string $id = ''` | — | Конструктор класса. Вызывает родительский конструктор `Div`, добавляет CSS-класс `'Dialog'` через `addclass()`. |

### Методы настройки отображения

| Метод | Параметры | Возвращает | Описание |
|-------|-----------|------------|----------|
| **capt** | `string $str` | `$this` | Устанавливает заголовок диалога. Устанавливает атрибут `title` в значение `$str`. Используется jQuery UI Dialog для отображения заголовка окна. Поддерживает fluent interface. |
| **autoopen** | `bool $open = true` | `$this` | Устанавливает параметр `autoOpen` диалога. Если `true` — диалог открывается автоматически при инициализации. Если `false` — требует явного вызова `open()`. Поддерживает fluent interface. |
| **modal** | `bool $val = true` | `$this` | Устанавливает модальный режим диалога. Если `true` — блокирует взаимодействие с остальной страницей пока диалог открыт. Сохраняет в секции `'options'`. Поддерживает fluent interface. |
| **show** | `string $effect = 'blind'`, `int $duration = 1000` | `$this` | Устанавливает эффект показа диалога. Принимает имя эффекта jQuery UI (`'blind'`, `'fade'`, `'slide'` и др.) и длительность в миллисекундах. Сохраняет в секции `'options'` как массив `['effect' => $effect, 'duration' => $duration]`. Поддерживает fluent interface. |
| **hide** | `string $effect = 'explode'`, `int $duration = 1000` | `$this` | Устанавливает эффект скрытия диалога. Принимает имя эффекта jQuery UI и длительность в миллисекундах. Сохраняет в секции `'options'` как массив `['effect' => $effect, 'duration' => $duration]`. Поддерживает fluent interface. |
| **option** | `string $name`, `mixed $val = null` | `mixed/$this` | Универсальный метод управления параметрами диалога. Если `$val === null` — возвращает значение параметра из секции `'options'`. Если передано — устанавливает параметр. Поддерживает fluent interface. |

### Методы позиционирования

| Метод | Параметры | Возвращает | Описание |
|-------|-----------|------------|----------|
| **left** | `mixed $val` | `$this` | Устанавливает позицию диалога по горизонтали. Может быть числом (пиксели) или строкой (`'center'`, `'left'`, `'right'`). Сохраняет в секции `'options'`. Поддерживает fluent interface. |
| **top** | `mixed $val` | `$this` | Устанавливает позицию диалога по вертикали. Может быть числом (пиксели) или строкой (`'center'`, `'top'`, `'bottom'`). Сохраняет в секции `'options'`. Поддерживает fluent interface. |
| **width** | `mixed $val` | `$this` | Устанавливает ширину диалога. Может быть числом (пиксели) или строкой (`'auto'`, `'500px'`). Сохраняет в секции `'options'`. Поддерживает fluent interface. |
| **height** | `mixed $val` | `$this` | Устанавливает высоту диалога. Может быть числом (пиксели) или строкой (`'auto'`, `'300px'`). Сохраняет в секции `'options'`. Поддерживает fluent interface. |

### Методы работы с кнопками

| Метод | Параметры | Возвращает | Описание |
|-------|-----------|------------|----------|
| **button** | `string $caption`, `string $onClick = ''` | `$this` | Добавляет кнопку действия в диалог. Принимает текст кнопки и JavaScript-код обработчика нажатия. Сохраняет в секции `'buttons'` с ключом `$caption`. При компиляции генерирует методы `button1()`, `button2()` и т.д. для каждого обработчика. Поддерживает fluent interface. |

### Жизненный цикл

| Метод | Параметры | Возвращает | Описание |
|-------|-----------|------------|----------|
| **compile** | — | — | Генерация клиентских методов LTS-объекта. Извлекает кнопки из секции `'buttons'`, генерирует обработчики нажатий (`button1()`, `button2()` и т.д.). Преобразует массив кнопок в формат jQuery UI Dialog. Извлекает параметры из секции `'options'`, кодирует через `JS::paramobject()`. Добавляет JS-код инициализации в область `'ready'`: устанавливает `buttonN` методы, создает jQuery-объект диалог. Генерирует клиентские методы `open()` и `close()` через `compilemethod()`. |

### Клиентские методы LTS-объекта

| Метод | Параметры | Возвращает | Описание |
|-------|-----------|------------|----------|
| **LTS(dialog).open()** | — | `this` | Открывает диалоговое окно. Использует jQuery UI Dialog метод `dialog('open')`. Поддерживает цепочечные вызовы. |
| **LTS(dialog).close()** | — | `this` | Закрывает диалоговое окно. Использует jQuery UI Dialog метод `dialog('close')`. Поддерживает цепочечные вызовы. |
| **LTS(dialog).buttonN()** | — | — | Обработчики кнопок действия. Генерируются автоматически при компиляции для каждой кнопки через `button()`. Вызывают JavaScript-код переданный в `$onClick`. Методы нумеруются последовательно: `button1()`, `button2()`, `button3()` и т.д. Поддерживает цепочечные вызовы. |

## Примеры использования

```php
// Создание диалогового окна
$dialog = LTS::Dialog();
$dialog->capt('Подтверждение действия')
    ->modal(true)
    ->width(400)
    ->height(200)
    ->button('ОК', "alert('Действие выполнено')")
    ->button('Отмена', "LTS(dialog).close()")
    ->autoopen(false);

// Настройка анимации
$dialog->show('slide', 500)
    ->hide('explode', 1000);
```

## Примечания
*   Класс расширяет `Div`, поэтому наследует все методы Flexbox (`flex()`, `row()`, `column()`, `content()`, `align()`, `gap()` и др.), управления потомками (`add()`, `del()`, `getchilds()`, `move()`) и стилизации (`css()`, `width()`, `height()`).
*   Класс расширяет `Element`, `Quark`, `Construct`, поэтому поддерживает полный жизненный цикл объектов: `compile()`, `shine()`, `childs()`, `addinspace()`, а также хуки (`$check`, `$before`, `$on` и др.).
*   **Зависимость от jQuery UI:** Диалог использует jQuery UI Dialog — требуется подключение `jquery-ui.css` и `jquery-ui.js` в пространстве `Space`.
*   Все методы настройки диалога возвращают `$this` для поддержки fluent interface и цепочечных вызовов.
*   Метод `capt()` устанавливает атрибут `title` — jQuery UI Dialog использует его для заголовка окна.
*   Метод `autoopen()` по умолчанию `true` — диалог открывается сразу после инициализации. Для отложенного открытия установить в `false` и вызывать `LTS(dialog).open()`.
*   Эффекты `show()` и `hide()` используют jQuery UI Effects — доступны: `'blind'`, `'bounce'`, `'clip'`, `'drop'`, `'explode'`, `'fade'`, `'fold'`, `'highlight'`, `'puff'`, `'pulsate'`, `'scale'`, `'shake'`, `'slide'`, `'transfer'`.
*   Длительность эффектов указывается в миллисекундах — по умолчанию `1000` (1 секунда).
*   Метод `button()` добавляет кнопки в секцию `'buttons'` — при компиляции преобразуются в формат jQuery UI Dialog.
*   Обработчики кнопок нумеруются последовательно — `button1()`, `button2()`, `button3()` — порядок определяется порядком добавления через `button()`.
*   JavaScript-код обработчиков кнопок передаётся как строка — может содержать любой валидный JS.
*   Метод `modal()` блокирует взаимодействие с страницей — полезно для подтверждений и критических действий.
*   Позиционирование через `left()` и `top()` — может быть числом (пиксели от края окна) или строкой (`'center'`, `'left'`, `'right'`, `'top'`, `'bottom'`).
*   Размеры через `width()` и `height()` — может быть числом (пиксели), строкой (`'500px'`, `'auto'`) или процентом (`'80%'`).
*   Метод `option()` — универсальный доступ к параметрам jQuery UI Dialog — можно установить любые поддерживаемые опции.
*   Метод `compile()` генерирует 2 клиентских метода: `open()` и `close()` — доступны через `LTS.get('{id}')` и `LTS(var)`, где var - PHP-переменная, ссылкающаяся на экземпляр класса Dialog.
*   Для кастомизации внешнего вида — использовать CSS-класс `.Dialog` и классы jQuery UI Dialog (`ui-dialog`, `ui-dialog-titlebar`, `ui-dialog-content` и др.).
*   События jQuery UI Dialog (`beforeClose`, `close`, `create`, `drag`, `dragStart`, `dragStop`, `focus`, `open`, `resize`, `resizeStart`, `resizeStop`) — можно назначать через `option('eventName', handler)`.
*   Для получения значения параметра — использовать `option('name')` без второго аргумента.
*   Метод `button()` с пустым `$onClick` — создаёт кнопку без обработчика — можно назначить позже через клиентский JS.
*   Диалог поддерживает перетаскивание — по умолчанию включено, можно отключить через `option('draggable', false)`.
*   Диалог поддерживает изменение размера — по умолчанию включено, можно отключить через `option('resizable', false)`.
*   Для модальных диалогов — фон затемняется через jQuery UI Overlay — можно кастомизировать через CSS.
*   Закрытие по клику вне диалога — отключено по умолчанию для модальных, можно включить через `option('closeOnEscape', true)`.
*   Закрытие по клавише Escape — управляется через `option('closeOnEscape', true/false)`.
*   Для программного закрытия — использовать `LTS(dialog).close()` — триггерит событие `close` jQuery UI.
*   Для программного открытия — использовать `LTS(dialog).open()` — триггерит событие `open` jQuery UI.
*   События диалога можно обрабатывать через хуки LOTIS — `check`, `before`, `on` для методов `open()` и `close()`.
*   Диалог автоматически центрируется при `'center'` позиционировании — jQuery UI вычисляет позицию относительно окна.
*   Минимальная ширина/высота — контролируется через `option('minWidth', value)` и `option('minHeight', value)`.
*   Максимальная ширина/высота — контролируется через `option('maxWidth', value)` и `option('maxHeight', value)`.
*   Для запрета закрытия — использовать `option('closeOnEscape', false)` и не добавлять кнопку закрытия.
*   Кнопка закрытия (крестик) — отображается по умолчанию — можно скрыть через CSS `.ui-dialog-titlebar-close { display: none; }`.
*   Для динамического изменения заголовка — использовать `$(var).dialog('option', 'title', 'New Title')`.
*   Для динамического изменения размеров — использовать `$(var).dialog('option', 'width', 600)`.
*   Для проверки состояния — использовать `$(var).dialog('isOpen')` — возвращает `true`/`false`.
*   Для отложенной инициализации — установить `autoopen(false)` и вызывать `open()` вручную.
*   Для очистки содержимого — использовать `$(var).empty()` перед добавлением нового контента.
