# Класс Video

## Назначение
Класс представления видео-плеера фреймворка LOTIS, наследующий `Element`. Предназначен для создания HTML5-видео элементов с поддержкой множественных источников, автоматическим определением MIME-типов по расширению файла и настройкой параметров воспроизведения. Класс предоставляет методы для управления автовоспроизведением, предзагрузкой, элементами управления, зацикливанием, постером-превью и другими атрибутами видео-плеера. Поддерживает форматы MP4, WebM, OGV и MP3.

### Основные возможности

* **Воспроизведение видео** и аудио файлов
* **Автоматическое определение** MIME-типов
* **Множественные источники** для кроссбраузерной совместимости
* **Управление воспроизведением** через методы
* **Настройка внешнего вида** и поведения

### Поддерживаемые форматы

* **MP4** — видео/аудио с кодеками H.264 и AAC
* **WebM** — видео с кодеками VP8 и Vorbis
* **OGV** — видео с кодеками Theora и Vorbis
* **MP3** — аудиофайлы

## Наследуемые свойства

| Свойство | Тип | Видимость | Описание |
|----------|-----|-----------|----------|
| **$id** | `string` | `public` | Наследуется от `Construct`. Уникальный идентификатор объекта, используется как HTML-атрибут `id`. |
| **$type** | `string` | `public` | Наследуется от `Element`. Устанавливается в `'html'`. Определяет тип объекта для фильтрации и обработки. |
| **$tagname** | `string` | `public` | Наследуется от `Element`. Устанавливается в `'video'` в конструкторе. Определяет HTML-тег элемента при рендеринге. |
| **$classname** | `string` | `public` | Наследуется от `Element`. CSS-классы элемента, управляются методами `addclass()`, `removeclass()`, `hasclass()`. |
| **$caption** | `string` | `public` | Наследуется от `Element`. Текстовое содержимое элемента. Для видео используется как альтернативный текст. |

## Методы

### Конструктор

| Метод | Параметры | Возвращает | Описание |
|-------|-----------|------------|----------|
| **__construct** | `string $id = ''` | — | Конструктор класса. Вызывает родительский конструктор `Element`, устанавливает `tagname` в `'video'`. |

### Методы добавления источников

| Метод | Параметры | Возвращает | Описание |
|-------|-----------|------------|----------|
| **src** | `string $src` | `$this` | Добавляет источник медиафайла. Создаёт элемент `<source>` с атрибутом `src`. Автоматически определяет MIME-тип по расширению файла (mp4, webm, ogv, mp3). Для mp3 устанавливает `audio/mpeg`, для mp4 — `video/mp4` с кодеками, для webm — `video/webm`, для ogv — `video/ogg`. Для других форматов тип не указывается. Поддерживает fluent interface. |
| **sources** | `array $sources` | `$this` | Добавляет несколько источников видео. Принимает массив URL-адресов, для каждого вызывает `src()`. Полезно для предоставления альтернативных форматов для разных браузеров. Поддерживает fluent interface. |

### Методы настройки воспроизведения

| Метод | Параметры | Возвращает | Описание |
|-------|-----------|------------|----------|
| **autoplay** | — | `$this` | Включает автовоспроизведение видео. Устанавливает атрибут `autoplay="autoplay"`. Поддерживает fluent interface. |
| **preload** | — | `$this` | Включает предзагрузку видео. Устанавливает атрибут `preload="auto"`. Поддерживает fluent interface. |
| **loop** | — | `$this` | Включает бесконечное воспроизведение. Устанавливает атрибут `loop="loop"`. Поддерживает fluent interface. |
| **muted** | — | `$this` | Устанавливает звук по умолчанию в беззвучный режим. Устанавливает атрибут `muted="muted"`. Требуется для автовоспроизведения в большинстве браузеров. Поддерживает fluent interface. |
| **playsinline** | — | `$this` | Разрешает воспроизведение без перехода в полноэкранный режим на мобильных устройствах. Устанавливает атрибут `playsinline="playsinline"`. Поддерживает fluent interface. |

### Методы настройки внешнего вида

| Метод | Параметры | Возвращает | Описание |
|-------|-----------|------------|----------|
| **poster** | `string $src = null` | `$this` | Устанавливает изображение-превью (постер), отображаемое до начала воспроизведения. Устанавливает атрибут `poster`. Поддерживает fluent interface. |
| **controls** | — | `$this` | Отображает элементы управления плеером (play, pause, громкость и др.). Устанавливает атрибут `controls="controls"`. Поддерживает fluent interface. |
| **width** | `mixed $val = null` | `$this` | Устанавливает ширину видео. Может быть числом (пиксели) или строкой (проценты, `auto`). Устанавливает атрибут `width`. Поддерживает fluent interface. |
| **height** | `mixed $val = null` | `$this` | Устанавливает высоту видео. Может быть числом (пиксели) или строкой (проценты, `auto`). Устанавливает атрибут `height`. Поддерживает fluent interface. |

### Методы безопасности и совместимости

| Метод | Параметры | Возвращает | Описание |
|-------|-----------|------------|----------|
| **crossorigin** | `string $value = 'anonymous'` | `$this` | Устанавливает режим загрузки видео с других доменов (CORS). По умолчанию `'anonymous'`. Устанавливает атрибут `crossorigin`. Поддерживает fluent interface. |
| **disablepictureinpicture** | — | `$this` | Отключает режим «картинка в картинке». Устанавливает атрибут `disablepictureinpicture="true"`. Поддерживает fluent interface. |

## Примеры использования

```php
// Создание видеоэлемента
$video = LTS::Video();

// Добавление источников
$video->src('video.mp4')
      ->src('video.webm')
      ->src('video.ogv');

// Настройка параметров
$video->controls()
      ->autoplay()
      ->loop()
      ->width(640)
      ->height(480)
      ->poster('preview.jpg');

// Дополнительные настройки для мобильных устройств
$video->playsinline()
      ->crossorigin();
```

## Примечания
*   Класс расширяет `Element`, поэтому наследует все методы управления потомками, стилями, атрибутами и событиями.
*   Класс расширяет `Quark` и `Construct`, поэтому поддерживает полный жизненный цикл объектов: `compile()`, `shine()`, `childs()`, `addinspace()`, а также хуки (`$check`, `$before`, `$on` и др.).
*   **MIME-типы:** Метод `src()` автоматически определяет тип контента по расширению файла: mp4 → `video/mp4`, webm → `video/webm`, ogv → `video/ogg`, mp3 → `audio/mpeg`.
*   **Несколько источников:** Метод `sources()` позволяет добавить несколько форматов видео для совместимости с разными браузерами — браузер выберет первый поддерживаемый формат.
*   **Автовоспроизведение:** Для работы `autoplay` в современных браузерах требуется также установить `muted()` — большинство браузеров блокируют автовоспроизведение со звуком.
*   **Элементы управления:** Атрибут `controls` добавляет стандартный интерфейс плеера (play/pause, громкость, прогресс, полноэкранный режим).
*   **Постер:** Изображение `poster` отображается до начала воспроизведения — полезно для превью и улучшения UX.
*   **Мобильные устройства:** Атрибут `playsinline` предотвращает автоматический переход в полноэкранный режим на iOS и Android.
*   **CORS:** Атрибут `crossorigin` требуется для загрузки видео с других доменов — по умолчанию `'anonymous'`.
*   **Picture-in-Picture:** Атрибут `disablepictureinpicture` отключает режим плавающего окна — полезен для кастомных интерфейсов.
*   **Источники:** Каждый вызов `src()` создаёт отдельный элемент `<source>` внутри `<video>` — браузер выбирает первый поддерживаемый.
*   **Атрибуты:** Все методы устанавливают HTML-атрибуты через `attr()` — значения экранируются при рендеринге.
*   **Fluent interface:** Все методы возвращают `$this` для поддержки цепочечных вызовов.
*   **Потомки:** Элементы `<source>` добавляются в потомки через `add()` — рендерятся внутри тега `<video>`.
*   **Рендеринг:** При `shine()` создаётся HTML-тег `<video>` с атрибутами и вложенными элементами `<source>`.
*   **Пространство имён:** Регистрируется через `addinspace()` — доступно в пространстве `Space`.
*   **События:** Можно назначать обработчики событий через `on()` — `play`, `pause`, `ended`, `error` и др.
*   **Стили:** CSS-свойства добавляются через `css()` — для кастомизации внешнего вида плеера.
*   **Классы:** CSS-классы добавляются через `addclass()` — для стилизации через внешние файлы.
*   **ID:** Уникальный идентификатор используется для связи с клиентским объектом через `LTS.get('{id}')`.
*   **Предзагрузка:** `preload="auto"` начинает загрузку видео сразу — может влиять на производительность.
*   **Зацикливание:** `loop` воспроизводит видео повторно после завершения — полезно для фоновых видео.
*   **Ширина/Высота:** Атрибуты `width` и `height` задают размеры плеера — рекомендуется для предотвращения сдвига макета.
*   **Беззвучный режим:** `muted` отключает звук по умолчанию — требуется для автовоспроизведения.
*   **Кросс-домен:** `crossorigin` требуется для доступа к видео через JavaScript — например, для кастомных контролов.
*   **Picture-in-Picture:** Отключается через `disablepictureinpicture` — для кастомных интерфейсов.
*   **Форматы:** Поддерживает mp4 (H.264), webm (VP8), ogv (Theora), mp3 (audio).
*   **Кодеки:** Для mp4 указываются кодеки `avc1.42E01E, mp4a.40.2` — для совместимости.
*   **Альтернативные форматы:** Рекомендуется добавлять несколько источников через `sources()` — для максимальной совместимости.
