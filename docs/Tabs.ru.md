# Класс Tabs

## Назначение
Класс вкладочного интерфейса фреймворка LOTIS, наследующий `Div`. Предназначен для создания переключателей между секциями контента с использованием виджета jQuery UI Tabs. Позволяет группировать содержимое в панели, где одновременно отображается только одна активная вкладка. Поддерживает горизонтальное и вертикальное расположение вкладок, сворачивание панелей, активацию по клику или наведению мыши. Класс автоматически генерирует структуру HTML для вкладок (список `<ul>` с заголовками и `<div>` с содержимым), создаёт клиентский JavaScript-код для инициализации виджета и предоставляет методы для программного переключения между вкладками.

### Основные возможности

* **Горизонтальное** и **вертикальное** расположение вкладок
* **Сворачиваемые** вкладки
* **События** наведения мыши
* **Динамическое** добавление вкладок

## Свойства

| Свойство | Тип | Видимость | Описание |
|----------|-----|-----------|----------|
| **$vertical** | `bool` | `public` | Флаг вертикального расположения вкладок. По умолчанию `false` (горизонтально). Управляется методом `vertical()`. |
| **$ul** | `Element` | `public` | Объект-контейнер для заголовков вкладок. Создаётся в конструкторе как элемент `<ul>`, добавляется в потомки. Используется jQuery UI Tabs для навигации. |

### Наследуемые свойства

| Свойство | Тип | Видимость | Описание |
|----------|-----|-----------|----------|
| **$id** | `string` | `public` | Наследуется от `Construct`. Уникальный идентификатор объекта, используется как HTML-атрибут `id`. |
| **$tagname** | `string` | `public` | Наследуется от `Div`. По умолчанию `'div'`. Определяет HTML-тег элемента при рендеринге. |
| **$classname** | `string` | `public` | Наследуется от `Element`. Автоматически добавляется класс `'Tabs'` в конструкторе. |

## Методы

### Конструктор

| Метод | Параметры | Возвращает | Описание |
|-------|-----------|------------|----------|
| **__construct** | `string $id = ''` | — | Конструктор класса. Вызывает родительский конструктор `Div`, добавляет CSS-класс `'Tabs'`, создаёт объект `$ul` типа `Element` с тегом `'ul'`, добавляет его в потомки. Устанавливает `$vertical` в `false`. |

### Методы создания вкладок

| Метод | Параметры | Возвращает | Описание |
|-------|-----------|------------|----------|
| **newtab** | `string $caption` | `Div` | Создаёт новую вкладку. Создаёт объект `Div` с классом `'Tab'`, устанавливает `$caption`, добавляет в потомки. Возвращает объект вкладки для добавления контента. |

### Методы настройки отображения

| Метод | Параметры | Возвращает | Описание |
|-------|-----------|------------|----------|
| **collaps** | — | `$this` | Включает режим сворачивания всех вкладок. Устанавливает опцию `'collapsible'` в `true`. Поддерживает fluent interface. |
| **onmouse** | — | `$this` | Включает активацию вкладок по наведению мыши. Устанавливает опцию `'event'` в `'mouseover'`. Поддерживает fluent interface. |
| **vertical** | `bool $v = true` | `$this` | Устанавливает вертикальное расположение вкладок. Сохраняет в `$vertical`. При компиляции генерирует JS-код для добавления классов `'ui-tabs-vertical'` и `'ui-helper-clearfix'`. Поддерживает fluent interface. |

### Жизненный цикл

| Метод | Параметры | Возвращает | Описание |
|-------|-----------|------------|----------|
| **compile** | — | — | Генерация клиентских JavaScript-методов. Находит все потомки с классом `'Tab'`, для каждого создаёт элемент `<li>` с ссылкой `<a>` (текст из `$caption`, href на ID вкладки). Очищает `$caption` у вкладок. Кодирует опции через `json_encode()`. Создаёт JS-объект с методами: `vertical()`, `horizontal()`. Добавляет JS-код инициализации в область `'ready'`: `jQuery('#{$this->id}').tabs({options})`. Если `$vertical === true` — добавляет вызов `vertical()`. Вызывает `parent::compile()` в конце. |

## Клиентские методы LTS-объекта

| Метод | Параметры | Возвращает | Описание |
|-------|-----------|------------|----------|
| **LTS(tabs).vertical()** | — | — | Переключает вкладки в вертикальный режим. Добавляет классы `'ui-tabs-vertical'`, `'ui-helper-clearfix'` контейнеру, меняет классы элементов `<li>` с `'ui-corner-top'` на `'ui-corner-left'`. |
| **LTS(tabs).horizontal()** | — | — | Переключает вкладки в горизонтальный режим. Удаляет классы `'ui-tabs-vertical'`, `'ui-helper-clearfix'`, меняет классы элементов `<li>` с `'ui-corner-left'` на `'ui-corner-top'`. |

## Примеры использования

```php
// Создание системы вкладок
$tabs = LTS::Tabs();

// Добавление вкладок
$tab1 = $tabs->newtab('Вкладка 1');
$tab2 = $tabs->newtab('Вкладка 2');

// Добавление контента во вкладки
$tab1->add(LTS::Html('p')->capt('Содержимое первой вкладки'));
$tab2->add(LTS::Html('p')->capt('Содержимое второй вкладки'));

// Настройка отображения
$tabs->vertical() // Вертикальное расположение
     ->collaps();    // Событие наведения мыши
```

## Примечания
*   Класс расширяет `Div`, поэтому наследует все методы Flexbox, управления потомками и стилизации.
*   Класс расширяет `Element`, `Quark`, `Construct`, поэтому поддерживает полный жизненный цикл объектов и хуки.
*   **Зависимость от jQuery UI:** Вкладки используют jQuery UI Tabs — требуется подключение `jquery-ui.css` и `jquery-ui.js` в пространстве `Space`.
*   Все методы настройки возвращают `$this` для поддержки fluent interface и цепочечных вызовов.
*   Вкладки создаются через `newtab()` — каждая вкладка получает класс `'Tab'` для идентификации при компиляции.
*   При компиляции для каждой вкладки создаётся заголовок `<li><a>` — текст берётся из `$tab->caption`, ссылка указывает на ID вкладки.
*   Заголовки добавляются в `$ul` — это стандартная структура для jQuery UI Tabs.
*   Метод `compile()` генерирует 2 клиентских метода: `vertical()`, `horizontal()` — доступны через `LTS.get('{id}')`.
*   Опции вкладок кодируются через `json_encode()` с `JSON_UNESCAPED_UNICODE` — кириллица в заголовках сохраняется без экранирования.
*   Вертикальный режим переключается через CSS-классы — требует дополнительной стилизации через jQuery UI CSS.
*   Активация по наведению мыши устанавливается через опцию `'event'` — по умолчанию активация по клику.
*   Режим сворачивания позволяет скрыть все вкладки — по умолчанию в jQuery UI одна вкладка всегда открыта.
*   Для кастомизации внешнего вида — использовать CSS-классы `.Tabs`, `.Tab`, `.ui-tabs`.
*   Методы `vertical()` и `horizontal()` на клиенте меняют классы — не пересоздают виджет.
*   Класс не реализует анимации переключения — используются стандартные анимации jQuery UI.
*   Вкладки могут содержать любые дочерние элементы — `Div`, `Form`, `Input`, `DataTable` и др.
*   При компиляции вкладки с пустым `$caption` — заголовок создаётся пустым.
*   Метод `collaps()` устанавливает опцию `'collapsible'` — может комбинироваться с другими опциями.
*   Метод `onmouse()` устанавливает опцию `'event'` — переопределяет стандартное поведение по клику.
*   При компиляции `$tab->caption = ''` — очищается после создания заголовка.
*   Для сохранения заголовка — использовать отдельное свойство.
*   Вкладки совместимы с `Dialog` — могут размещаться внутри модального окна.
*   Для горизонтального режима по умолчанию — не вызывать `vertical()`.
*   При компиляции `$optionsstr` — кодируется через `json_encode()` с флагом `JSON_UNESCAPED_UNICODE`.
*   Метод `newtab()` возвращает `Div` — позволяет цепочечное добавление контента.
*   Для скрытия вкладок — использовать `css()->add('display', 'none')`.
*   Вкладки интегрируются с системой сигналов — можно отправлять сигналы при переключении.
*   Для многоязычных вкладок — использовать `$lang->say()` в `$caption`.
*   При компиляции заголовки `<li><a>` создаются автоматически — не требуют явного добавления.
