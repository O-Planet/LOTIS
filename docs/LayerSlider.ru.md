# Класс LayerSlider

## Назначение
Компонент переключения слоёв (слайдер) фреймворка LOTIS, наследующий `Div`. Предназначен для организации интерфейса с поочередным отображением содержимого (слоёв), где в каждый момент времени виден только один активный слой. Поддерживает программное переключение (методы `next`, `prev`, `goto`), жесты свайпа (мышь/тач), зацикливание перехода и визуальный индикатор текущей позиции. Используется для пошаговых форм, галерей, переключения вкладок интерфейса и мастер-процессов.

### Основные возможности

* **Навигация между слоями** контента
* **Поддержка свайпов** для переключения
* **Индикаторы** текущего слоя
* **Настройка отображения** активного слоя
* **Циклическое воспроизведение**

## Свойства

| Свойство | Тип | Видимость | Описание |
|----------|-----|-----------|----------|
| **$layers** | `array` | `private` | Массив идентификаторов потомков, распознанных как слои. Заполняется автоматически методом `add()`. |
| **$captions** | `array` | `private` | Массив заголовков слоёв. Используется для генерации подписей индикатора. Заполняется из `$child->caption` при добавлении слоя. |
| **$loop** | `bool` | `private` | Флаг зацикливания переключения. Если `true` — переход с последнего слоя на первый и наоборот. По умолчанию `true`. |
| **$swipeSensitivity** | `int` | `private` | Минимальное смещение курсора/пальца в пикселях для регистрации свайпа. По умолчанию `50`. |
| **$displayStyle** | `string` | `private` | CSS-стиль отображения активного слоя. По умолчанию `'block'`. Допустимые: `'block'`, `'flex'`, `'inline-block'`, `'grid'`. |
| **$activeLayer** | `mixed` | `private` | Изначально активный слой. Может быть объектом слоя, его ID (строка) или номером (int, 1-based). По умолчанию `1` (первый слой). |

### Наследуемые свойства

| Свойство | Тип | Видимость | Описание |
|----------|-----|-----------|----------|
| **$id** | `string` | `public` | Наследуется от `Construct`. Уникальный идентификатор слайдера. |
| **$tagname** | `string` | `public` | Наследуется от `Div`. По умолчанию `'div'`. |
| **$classname** | `string` | `public` | Наследуется от `Element`. Автоматически добавляется класс `'LayerSlider'`. |
| **$storage** | `array` | `public` | Наследуется от `Quark`. Хранилище потомков и метаданных. |

## Методы

### Конструктор

| Метод | Параметры | Возвращает | Описание |
|-------|-----------|------------|----------|
| **__construct** | `string $id = ''` | — | Конструктор класса. Вызывает родительский конструктор `Div`, добавляет класс `'LayerSlider'`, устанавливает CSS `position: relative` и `overflow: hidden`. |

### Добавления контента

| Метод | Параметры | Возвращает | Описание |
|-------|-----------|------------|----------|
| **add** | `mixed $name`, `mixed $value = null` | `mixed` | Переопределённый метод добавления потомка. Если добавляется объект типа `html` с тегом `div` — автоматически добавляет класс `'Layer'`, сохраняет ID в `$layers`, заголовок в `$captions`. Вызывает родительский `add()`. |

### Методы настройки

| Метод | Параметры | Возвращает | Описание |
|-------|-----------|------------|----------|
| **active** | `mixed $child` | `$this` | Устанавливает изначально активный слой. Принимает объект слоя, ID строки или номер (1-based). Сохраняет в `$activeLayer`. Поддерживает fluent interface. |
| **loop** | `bool $enable = true` | `$this` | Включает или выключает зацикливание переключения слоёв. Устанавливает `$loop`. Поддерживает fluent interface. |
| **sensitivity** | `int $px` | `$this` | Устанавливает чувствительность свайпа в пикселях. Устанавливает `$swipeSensitivity`. Поддерживает fluent interface. |
| **setdisplay** | `string $style = 'block'` | `$this` | Устанавливает CSS-стиль отображения активного слоя (`display`). Проверяет допустимые значения. Устанавливает `$displayStyle`. Поддерживает fluent interface. |
| **createIndicator** | `string $css = null` | `Div` | Создаёт визуальный индикатор (точки) для переключения слоёв. Создаёт объект `Div` с классом `'layer-indicator'`, генерирует точки-кнопки для каждого слоя. Генерирует JS-метод `updateIndicator()` для подсветки активной точки. Возвращает объект индикатора. |

### Жизненный цикл

| Метод | Параметры | Возвращает | Описание |
|-------|-----------|------------|----------|
| **compile** | — | — | Генерация клиентских JavaScript-методов. Определяет индекс активного слоя. Скрывает неактивные слои через CSS (`display: none`), активный — через `$displayStyle`. Генерирует методы: `next()`, `prev()`, `goto(index)`, `_fadeTransition(from, to)`. Добавляет обработчики событий `mousedown`/`touchstart` и `mouseup`/`touchend` для реализации свайпа. Блокирует выделение текста (`selectstart`). Вызывает `parent::compile()`. |

## Клиентские методы LTS-объекта

| Метод | Параметры | Возвращает | Описание |
|-------|-----------|------------|----------|
| **LTS(slider).next()** | — | — | Переключает на следующий слой. Если `$loop == true` — после последнего идёт первый. Запускает анимацию `_fadeTransition()`. Вызывает `updateIndicator()`. |
| **LTS(slider).prev()** | — | — | Переключает на предыдущий слой. Если `$loop == true` — перед первым идёт последний. Запускает анимацию `_fadeTransition()`. Вызывает `updateIndicator()`. |
| **LTS(slider).goto(index)** | `int index` | — | Переключает на слой по индексу (0-based). Проверяет границы массива `$layers`. Запускает анимацию `_fadeTransition()`. Вызывает `updateIndicator()`. |
| **LTS(slider)._fadeTransition(fromIndex, toIndex)** | `int fromIndex`, `int toIndex` | — | Внутренний метод анимации. Плавно меняет прозрачность (`opacity`) от 1 до 0 для старого слоя и от 0 до 1 для нового. Использует jQuery `animate()` длительностью 300ms. |
| **LTS(slider).updateIndicator()** | — | — | Обновляет состояние индикатора (если создан). Подсвечивает активную точку (меняет `background`). Вызывается автоматически при переключении. |

## Примеры использования

```php
// Создание слайдера
$slider = LTS::LayerSlider();

// Добавление слоев
$slider->add(LTS::Div('layer1')->capt('Контент слоя 1'))
       ->add(LTS::Div('layer2')->capt('Контент слоя 2'))
       ->add(LTS::Div('layer3')->capt('Контент слоя 3'));

// Настройка параметров
$slider->active('layer2')
       ->loop(true)
       ->sensitivity(50)
       ->setdisplay('flex');

// Создание индикатора
$slider->createIndicator();
```

## Примечания
*   Класс расширяет `Div`, поэтому наследует все методы Flexbox, управления потомками и стилизации.
*   Класс расширяет `Element`, `Quark`, `Construct`, поэтому поддерживает полный жизненный цикл объектов и хуки.
*   **Распознавание слоёв:** Только потомки типа `html` с тегом `'div'` автоматически считаются слоями (добавляется класс `'Layer'`, ID сохраняется в `$layers`). Остальные потомки игнорируются слайдером.
*   **Позиционирование:** Все слои автоматически получают CSS `position: absolute`, `top: 0`, `left: 0`, `width: 100%` — накладываются друг на друга.
*   **Анимация:** Переключение реализовано через изменение `opacity` (fade-эффект) длительностью 300ms через jQuery `animate()`.
*   **Свайп:** Обработчики `mousedown`/`touchstart` и `mouseup`/`touchend` регистрируются на контейнере слайдера. Свайп регистрируется если `deltaX > sensitivity` и `deltaY < 50`.
*   **Индикатор:** Метод `createIndicator()` создаёт отдельный `Div` с точками. Каждая точка генерирует клик с вызовом `LTS(slider).goto(index)`.
*   **Зацикливание:** Управляется флагом `$loop`. Если `false` — методы `next()` и `prev()` останавливаются на границах массива.
*   **Активный слой:** Индекс активного слоя вычисляется в `compile()` на основе `$activeLayer` (объект, ID или номер). По умолчанию активен первый слой (индекс 0).
*   **Стили:** Базовый класс `'LayerSlider'` получает `position: relative`, `overflow: hidden`. Класс `'Layer'` добавляется всем слоям автоматически.
*   **Зависимость:** Требуется jQuery для анимации (`animate()`) и обработки событий.
*   **Генерация JS:** Код инициализации и методы генерируются в области `'ready'` через `js('ready')`.
*   **Блокировка выделения:** Добавлен обработчик `selectstart` с возвратом `false` — предотвращает выделение текста при свайпе.
*   **Метод goto():** Принимает 0-based индекс. Если индекс вне диапазона — переключение не происходит.
*   **Метод updateIndicator():** Генерируется только если был вызван `createIndicator()`. Проверяет наличие элемента `#{$id}_indicator`.
*   **Скрытые слои:** Неактивные слои получают `display: none` при компиляции. Активный — `$displayStyle` (по умолчанию `block`).
*   **Контекст JS:** Методы `next`, `prev`, `goto` доступны через `LTS(layerslider)`.
*   **Чувствительность:** Параметр `$swipeSensitivity` по умолчанию 50px — может быть изменён методом `sensitivity()`.
*   **Отображение:** Параметр `$displayStyle` контролирует CSS `display` активного слоя — важно для flex/grid слоёв.
*   **Идентификация:** Слои идентифицируются по ID, которые сохраняются в массив `$layers` в порядке добавления.
*   **Заголовки:** Заголовки слоёв (`$caption`) используются для генерации `title` или подписей индикатора (в текущей реализации — только для массива `$captions`).
