# Класс Span

## Назначение
Класс представления строчного контейнера фреймворка LOTIS, наследующий `Element`. Предназначен для создания HTML-элемента `<span>`, который используется для группировки строчных элементов текста, применения стилей к отдельным фрагментам контента или добавления атрибутов к части текста без нарушения потоковой вёрстки. В отличие от `Div`, не создаёт блочный контейнер и не переносит содержимое на новую строку.

## Свойства

| Свойство | Тип | Видимость | Описание |
|----------|-----|-----------|----------|
| **$tagname** | `string` | `public` | Наследуется от `Element`. Устанавливается в `'span'` в конструкторе. Определяет HTML-тег элемента при рендеринге. |
| **$classname** | `string` | `public` | Наследуется от `Element`. CSS-классы элемента, управляются методами `addclass()`, `removeclass()`, `hasclass()`. |
| **$caption** | `string` | `public` | Наследуется от `Element`. Текстовое содержимое элемента. Отображается как текст внутри тега `<span>`. |
| **$storage** | `array` | `public` | Наследуется от `Quark`. Хранилище содержит секции `'childs'` (потомки), `'element'` (DOM-элемент), `'attr'` (HTML-атрибуты). |
| **$type** | `string` | `public` | Наследуется от `Element`. Устанавливается в `'html'`. Определяет тип объекта для фильтрации и обработки. |
| **$id** | `string` | `public` | Наследуется от `Construct`. Уникальный идентификатор объекта, используется как HTML-атрибут `id`. |
| **$space** | `Space` | `public` | Наследуется от `Construct`. Ссылка на пространство имён, куда добавляются сгенерированные элементы. |

## Методы

| Метод | Параметры | Возвращает | Описание |
|-------|-----------|------------|----------|
| **__construct** | `string $id = ''` | — | Конструктор класса. Вызывает родительский конструктор `Element`, устанавливает `tagname` в `'span'`. |

## Примечания
*   Класс расширяет `Element`, поэтому наследует все методы управления потомками, стилями, атрибутами и событиями (`css()`, `attr()`, `add()`, `on()` и др.).
*   Класс расширяет `Quark` и `Construct`, поэтому поддерживает полный жизненный цикл объектов: `compile()`, `shine()`, `childs()`, `addinspace()`, а также хуки.
*   **Строчный элемент:** По умолчанию не создаёт разрывов строки — используется для выделения части текста внутри абзаца или другого блочного элемента.
*   **Вложенность:** Может содержать другие элементы через метод `add()` — например, `Span`, `Html`, `Input` для сложной структуры текста.
*   **Стилизация:** CSS-свойства добавляются через `css()` — для изменения цвета, шрифта, отступов и других параметров текста.
*   **Идентификатор:** ID используется для связи с клиентским объектом через `LTS.get('{id}')` — для динамического изменения содержимого.
*   **Атрибуты:** Дополнительные атрибуты добавляются через `attr()` — `title`, `data-*`, `style` и др.
*   **Рендеринг:** При `shine()` создаётся HTML-тег `<span>` с атрибутами и содержимым (`caption` или потомки).
*   **Пространство имён:** Регистрируется через `addinspace()` — доступно в пространстве `Space` для сборки страницы.
*   **События:** Поддерживает назначение обработчиков событий через `on()` или `click()` — `mouseover`, `mouseout`, `click` и др.
*   **Хуки:** Поддерживает хуки жизненного цикла — `check`, `before`, `on` для внедрения логики создания.
*   **Потомки:** Потомки хранятся в `$storage['childs']` — рендерятся внутри тега `<span>`.
*   **Тип:** Устанавливается в `'html'` — для рендеринга в `Space` как HTML-элемент.
*   **Контекст:** Используется в текстовом контенте — для выделения, подсветки, форматирования фрагментов.
*   **Блочное поведение:** Не рекомендуется для блочной вёрстки — для этого используйте `Div` или `Grid`.
