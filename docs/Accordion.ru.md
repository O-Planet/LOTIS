# Класс Accordion

## Назначение
Класс аккордеона, наследующий `Div`. Предназначен для создания раскрывающихся панелей с использованием виджета jQuery UI Accordion. Позволяет группировать контент в секции, где одновременно открыта только одна панель. Класс автоматически преобразует дочерние элементы с классом `AccordionSection` в заголовки и содержимое аккордеона, генерирует клиентский JavaScript-код для инициализации виджета и предоставляет методы для программного управления состоянием панелей.

## Свойства

| Свойство | Тип | Видимость | Описание |
|----------|-----|-----------|----------|
| **$vertical** | `bool` | `public` | Флаг вертикального режима аккордеона. Управляется через опции jQuery UI. |
| **$ul** | `Element` | `public` | Объект-контейнер для заголовков и секций аккордеона. Создаётся в конструкторе, добавляется в потомки. |

### Наследуемые свойства

| Свойство | Тип | Видимость | Описание |
|----------|-----|-----------|----------|
| **$tagname** | `string` | `public` | Наследуется от `Div`. По умолчанию `'div'`. Определяет HTML-тег элемента при рендеринге. |
| **$classname** | `string` | `public` | Наследуется от `Element`. Автоматически добавляется класс `'Accordion'` в конструкторе. |
| **$storage** | `array` | `public` | Наследуется от `Quark`. Хранилище содержит секции `'childs'` (потомки-секции), `'element'` (DOM-элемент), `'options'` (параметры аккордеона). |
| **$id** | `string` | `public` | Наследуется от `Construct`. Уникальный идентификатор объекта, используется как HTML-атрибут `id`. |

## Методы

### Конструктор

| Метод | Параметры | Возвращает | Описание |
|-------|-----------|------------|----------|
| **__construct** | `string $id = ''` | — | Конструктор класса. Вызывает родительский конструктор `Div`, добавляет CSS-класс `'Accordion'` через `addclass()`, создаёт объект `$ul` типа `Element` и добавляет его в потомки. |

### Методы работы с секциями

| Метод | Параметры | Возвращает | Описание |
|-------|-----------|------------|----------|
| **newsection** | `string $caption` | `Div` | Создаёт новую секцию аккордеона. Создаёт объект `Div` с классом `'AccordionSection'`, устанавливает `$caption` и возвращает объект секции для добавления контента. |
| **collaps** | — | `$this` | Включает режим сворачивания всех панелей. Устанавливает опцию `'collapsible'` в `true`. Поддерживает fluent interface. |
| **active** | `int $numb = 1` | `$this` | Устанавливает активную панель по номеру. Номер передаётся 1-based, внутри преобразуется в 0-based для jQuery UI. Устанавливает опцию `'active'`. Поддерживает fluent interface. |
| **heightstyle** | `string $style = 'auto'` | `$this` | Устанавливает стиль высоты панелей. Допустимые значения: `'auto'`, `'fill'`, `'content'`. Устанавливает опцию `'heightStyle'`. Поддерживает fluent interface. |

### Методы жизненного цикла

| Метод | Параметры | Возвращает | Описание |
|-------|-----------|------------|----------|
| **compile** | — | — | Генерация клиентских JavaScript-методов. Находит все секции с классом `'AccordionSection'`, для каждой создаёт заголовок `<h3>` с текстом из `$caption`, перемещает секцию в `$ul`. Кодирует опции через `json_encode()`. Создаёт JS-объект с методами: `open(numb)`, `disable`, `enable`. Добавляет JS-код инициализации в область `'ready'`: `jQuery('#{$this->id}').accordion({options})`. Вызывает `parent::compile()` в конце. |

### Клиентские методы (генерируются при compile)

| Метод | Параметры | Возвращает | Описание |
|-------|-----------|------------|----------|
| **LTS(accordion).open(numb)** | `int numb` | — | Открывает панель по номеру. Вызывает `jQuery('#{id}').accordion('option', 'active', numb - 1)`. Номер 1-based. |
| **LTS(accordion).disable()** | — | — | Отключает аккордеон. Вызывает `jQuery('#{id}').accordion('disable')`. |
| **LTS(accordion).enable()** | — | — | Включает аккордеон. Вызывает `jQuery('#{id}').accordion('enable')`. |

### Пример использования

```php
// Создание аккордеона
$accordion = LTS::Accordion('myAccordion');

// Добавление секций
$section1 = $accordion->newsection('Секция 1');
$section2 = $accordion->newsection('Секция 2');

// Добавление контента
$section1->add(LTS::Div()->capt('Содержимое первой секции'));
$section2->add(LTS::Div()->capt('Содержимое второй секции'));

// Настройка параметров
$accordion->collaps()
    ->active(1)
    ->heightstyle('auto');

// Хук на предопределенный клиентский метод смены секции
$accordion->openhucks('open');
$accordion->method('onopen(numb)',
<<<JS
    console.log('Открыта секция ' + numb);
JS
);
```

## Примечания
*   Класс расширяет `Div`, поэтому наследует все методы Flexbox, управления потомками и стилизации.
*   Класс расширяет `Element`, `Quark`, `Construct`, поэтому поддерживает полный жизненный цикл объектов и хуки.
*   **Зависимость от jQuery UI:** Аккордеон использует jQuery UI Accordion — требуется подключение `jquery-ui.css` и `jquery-ui.js` в пространстве `Space`.
*   Все методы настройки возвращают `$this` для поддержки fluent interface и цепочечных вызовов.
*   Секции создаются через `newsection()` — каждая секция получает класс `'AccordionSection'` для идентификации при компиляции.
*   При компиляции для каждой секции создаётся заголовок `<h3>` — текст берётся из `$section->caption`, после чего `$caption` очищается.
*   Заголовки и секции добавляются в `$ul` — это контейнер для структуры аккордеона.
*   Метод `compile()` генерирует 3 клиентских метода: `open()`, `disable()`, `enable()` — доступны через `LTS.get('{id}')`.
*   Опции аккордеона кодируются через `json_encode()` с `JSON_UNESCAPED_UNICODE` — кириллица в заголовках сохраняется без экранирования.
*   Номер панели в `active()` и `open()` указывается 1-based — внутри преобразуется в 0-based для jQuery UI.
*   Режим `'collapsible'` позволяет свернуть все панели — по умолчанию в jQuery UI одна панель всегда открыта.
*   Стиль высоты `'auto'` — панели подстраиваются под содержимое, `'fill'` — заполняют доступное пространство, `'content'` — по высоте контента.
*   Для кастомизации внешнего вида — использовать CSS-классы `.Accordion`, `.AccordionSection`, `.ui-accordion`.
*   Метод `open()` на клиенте вызывает `accordion('option', 'active', ...)` — меняет активную панель без пересоздания.
*   Методы `disable()` и `enable()` блокируют/разблокируют взаимодействие — полезно для временного отключения.
*   Класс не реализует анимации открытия/закрытия — используются стандартные анимации jQuery UI.
*   Аккордеон может содержать любые дочерние элементы — `Div`, `Form`, `Input`, `DataTable` и др.
*   При компиляции секции с пустым `$caption` — заголовок создаётся пустым.
