# Класс Div

## Назначение
Универсальный контейнерный класс фреймворка LOTIS, наследующий `Element`. Предназначен для создания блочных элементов с полной поддержкой CSS Flexbox. Является основным строительным блоком для компоновки интерфейса — позволяет организовывать дочерние элементы в строки или колонки, управлять их выравниванием, отступами и порядком отображения. Класс предоставляет fluent interface для цепочечной настройки стилей и содержит готовые методы для распространённых layout-сценариев (rowbox, columnbox, gapCentered и др.). Все методы возвращают `$this` для поддержки последовательных вызовов.

### Основные возможности

* **Flexbox-макет** с полной поддержкой свойств
* **Различные типы отображения** (block, inline, flex)
* **Готовые пресеты** для типичных макетов
* **Управление отступами** между элементами

## Свойства

| Свойство | Тип | Видимость | Описание |
|----------|-----|-----------|----------|
| **$_reverse** | `bool` | `private` | Флаг обратного порядка элементов по главной оси. Устанавливается методом `reverse()`. Используется при генерации `flex-direction: row-reverse` или `column-reverse`. |
| **$_wrapreverse** | `bool` | `private` | Флаг обратного порядка строк при переносе. Устанавливается методом `wrapreverse()`. Используется при генерации `flex-wrap: wrap-reverse`. |

### Наследуемые свойства

| Свойство | Тип | Видимость | Описание |
|----------|-----|-----------|----------|
| **$id** | `string` | `public` | Наследуется от `Construct`. Уникальный идентификатор объекта, используется как HTML-атрибут `id`. |
| **$type** | `string` | `public` | Наследуется от `Element`. Устанавливается в `'html'`. Определяет тип объекта для фильтрации и обработки. |
| **$tagname** | `string` | `public` | Наследуется от `Element`. По умолчанию `'div'`. Определяет HTML-тег элемента при рендеринге. |
| **$classname** | `string` | `public` | Наследуется от `Element`. CSS-классы элемента, управляются методами `addclass()`, `removeclass()`, `hasclass()`. |
| **$caption** | `string` | `public` | Наследуется от `Element`. Текстовое содержимое контейнера. |
| **$storage** | `array` | `public` | Наследуется от `Quark`. Хранилище содержит секции `'childs'` (потомки), `'element'` (DOM-элемент), `'attr'` (атрибуты), `'options'` (опции). |

## Методы

### Конструктор

| Метод | Параметры | Возвращает | Описание |
|-------|-----------|------------|----------|
| **__construct** | `string $id = ''` | — | Конструктор класса. Вызывает родительский конструктор `Element`, инициализирует приватные флаги `$_reverse` и `$_wrapreverse` в `false`. |

### Типы отображения

| Свойство | Тип | Видимость | Описание |
|----------|-----|-----------|----------|
| **block** | — | `$this` | Устанавливает `display: block`. Блочный режим отображения. |
| **none** | — | `$this` | Устанавливает `display: none`. Скрывает элемент. |
| **inline** | — | `$this` | Устанавливает `display: inline`. Строчный режим отображения. |
| **flex** | — | `$this` | Устанавливает `display: flex`. Активирует Flexbox-режим для контейнера. |

### Готовые пресеты

| Свойство | Тип | Видимость | Описание |
|----------|-----|-----------|----------|
| **rowbox** | — | `$this` | Готовая конфигурация для горизонтального контейнера: `flex` + `row` + `wrap` + `content('center')` + `align('center')`. |
| **columnbox** | — | `$this` | Готовая конфигурация для вертикального контейнера: `flex` + `column` + `wrap(false)` + `content('center')` + `align('center')`. |
| **gapCentered** | — | `$this` | Готовая конфигурация: центрирует контейнер и устанавливает `gap: 10px`. Вызывает `content('center')`, `align('center')`, `gap('10px')`. |
| **gapStack** | — | `$this` | Готовая конфигурация для вертикального стека: `column()` + `gap('10px')`. |
| **gapInline** | — | `$this` | Готовая конфигурация для горизонтальной строки: `row()` + `gap('10px')`. |

### Flexbox-направление

| Свойство | Тип | Видимость | Описание |
|----------|-----|-----------|----------|
| **primaryaxis** | `string $axis` | `$this` | Устанавливает главную ось. Если `'row'` — вызывает `row()`, если `'column'` — вызывает `column()`. |
| **row** | — | `$this` | Устанавливает `flex-direction: row` или `row-reverse` (если активен `$_reverse`). Горизонтальная главная ось. |
| **column** | — | `$this` | Устанавливает `flex-direction: column` или `column-reverse` (если активен `$_reverse`). Вертикальная главная ось. |
| **wrap** | `bool $w = true` | `$this` | Устанавливает `flex-wrap`. Если `true` — `'wrap'` или `'wrap-reverse'` (если активен `$_wrapreverse`). Если `false` — `'nowrap'`. |
| **reverse** | `bool $r = true` | `$this` | Устанавливает флаг `$_reverse` и пересчитывает `flex-direction`. Меняет порядок элементов на противоположный по главной оси. |
| **wrapreverse** | `bool $r = true` | `$this` | Устанавливает флаг `$_wrapreverse` и пересчитывает `flex-wrap`. Меняет порядок строк на противоположный при переносе. |

### Flexbox-выравнивание

| Свойство | Тип | Видимость | Описание |
|----------|-----|-----------|----------|
| **content** | `string $justify` | `$this` | Устанавливает `justify-content`. Допустимые значения: `'start'`, `'end'`, `'center'`, `'between'`, `'around'`, `'evenly'`. Алиасы: `'beg'`, `'begin'` = `'start'`. |
| **align** | `string $align` | `$this` | Устанавливает `align-items`. Допустимые значения: `'start'`, `'end'`, `'center'`, `'stretch'`, `'baseline'`. Алиасы: `'beg'`, `'begin'` = `'start'`. |
| **aligncontent** | `string $cont` | `$this` | Устанавливает `align-content`. Применяется при разбиении на несколько строк. Значения: `'start'`, `'end'`, `'center'`, `'stretch'`, `'between'`, `'around'`. |
| **contentalign** | `string $cont` | `$this` | Алиас для `aligncontent()`. |
| **primarycontent** | `string $cont` | `$this` | Устанавливает выравнивание по главной оси. Обёртка над `content()`. |
| **secondalign** | `string $align` | `$this` | Устанавливает выравнивание по поперечной оси. Обёртка над `align()`. |
| **secondcontent** | `string $cont` | `$this` | Устанавливает выравнивание строк по поперечной оси. Обёртка над `aligncontent()`. |

### Управление отступами

| Свойство | Тип | Видимость | Описание |
|----------|-----|-----------|----------|
| **gap** | `string $value` | `$this` | Устанавливает CSS-свойство `gap`. Определяет отступ между всеми дочерними элементами. Пример: `'10px'`, `'1em 20px'`. |
| **rowGap** | `string $value` | `$this` | Устанавливает CSS-свойство `row-gap`. Вертикальный отступ между строками. |
| **columnGap** | `string $value` | `$this` | Устанавливает CSS-свойство `column-gap`. Горизонтальный отступ между колонками. |

### Проверка конфигурации

| Свойство | Тип | Видимость | Описание |
|----------|-----|-----------|----------|
| **isrow** | — | `bool` | Проверяет, является ли главная ось горизонтальной. Возвращает `true`, если `flex-direction` равен `'row'` или `'row_reverse'`. |
| **iscolumn** | — | `bool` | Проверяет, является ли главная ось вертикальной. Возвращает `true`, если `flex-direction` равен `'column'` или `'column_reverse'`. |

## Примеры использования

```php
// Создание базового контейнера
$div = LTS::Div('container')
    ->flex()
    ->row()
    ->gap('10px');

// Вертикальный контейнер
$column = LTS::Div()
    ->columnbox()
    ->width('100%');

// Контейнер с центрированием
$centered = LTS::Div()
    ->gapCentered()
    ->height('100vh');

// Пример с полным набором настроек
$layout = LTS::Div()
    ->flex()
    ->column()
    ->wrap(true)
    ->content('space-between')
    ->align('center')
    ->gap('20px');
```

## Примечания
*   Класс расширяет `Element`, поэтому наследует все методы управления потомками (`add()`, `del()`, `getchilds()`, `move()` и др.), стилями (`css()`, `width()`, `height()`), атрибутами (`attr()`, `removeattr()`) и событиями (`on()`, `click()`, `signal()`).
*   Класс расширяет `Quark` и `Construct`, поэтому поддерживает полный жизненный цикл объектов: `compile()`, `shine()`, `childs()`, `addinspace()`, а также хуки (`$check`, `$before`, `$on` и др.).
*   Все методы настройки Flexbox возвращают `$this` для поддержки fluent interface и цепочечных вызовов.
*   Приватные флаги `$_reverse` и `$_wrapreverse` влияют на генерацию CSS-свойств в методах `row()`, `column()`, `wrap()` — при изменении флага соответствующий метод вызывается повторно для пересчёта стиля.
*   Методы `rowbox()` и `columnbox()` предоставляют готовые пресеты для распространённых сценариев компоновки — центрирование содержимого по обеим осям с автоматическим переносом (или без).
*   Методы `gapCentered()`, `gapStack()`, `gapInline()` упрощают создание типовых layout-конструкций с отступами между элементами.
*   Алиасы значений (`'beg'`, `'begin'` = `'start'`) позволяют использовать более интуитивные названия при настройке выравнивания.
*   Методы `isrow()` и `iscolumn()` проверяют текущее значение `flex-direction` через объект CSS — возвращают актуальное состояние после всех изменений.
*   Свойство `gap` поддерживается современными браузерами для Flexbox-контейнеров — для старых браузеров может потребоваться полифил или альтернативная вёрстка.
*   Раздельные методы `rowGap()` и `columnGap()` позволяют независимо настраивать вертикальные и горизонтальные отступы.
*   Дочерние элементы добавляются через методы `add()`, `addmany()`, `addin()` — порядок добавления определяет порядок отображения в DOM (может быть изменён через `flex-direction: row-reverse` или `column-reverse`).
*   Класс не реализует собственные методы `shine()`, `compile()`, `childs()` — они наследуются от `Element` и `Quark` без изменений.
*   Для создания вложенных структур рекомендуется использовать цепочечные вызовы: `$div->add(LTS::Div()->columnbox()->add(...))`.
*   Метод `display()` является общим для всех значений — специализированные методы `block()`, `none()`, `inline()`, `flex()` предоставляют удобные обёртки для частых случаев.
