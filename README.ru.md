# LOTIS — PHP Framework for Fast Business Apps

"Меньше кода - больше смысла"

LOTIS (Low Time Script) — это фреймворк для быстрого создания интерактивных WEB-приложений и бизнес систем на PHP. 
Вы строите приложение как при десктопной разработке, используя принцпы ООП без привязки в верстке.   

## Ключевые преимущества

| Преимущество | Реализация |
|:---|:---|
| **Zero-config** | Работает сразу после подключения |
| **Единый язык** | PHP для сервера и клиентской генерации без ручной верстки |
| **Единое пространство имён** | Доступ к объектам через одни и те же переменные на сервере и на клиенте |
| **Предметная ориентированность** | Работа с бизнес-сущностями, а не DOM: документы с табличными частями, стоки и коллекторы данных, синхронизация с базой данных "на лету" |
| **Событийная модель** | Логика клиент-серверного взаимодействия без написания AJAX-запросов |
| **Сессионное состояние** | Автоматическая синхронизация глобальных переменных между сервером и клиентом |
| **Расширяемость** | Хуки, сигналы, пользовательские методы |
| **Минимум зависимостей** | Только PHP, jQuery, MySQL |


## Минимальный рабочий пример

```php
include_once 'newlotis/lotis.php';

$div = LTS::Div('main')
    ->columnbox();
$div->capt("Привет из LOTIS!")
    ->width('100%')
    ->height('100vh');
LTS::Space()->build($div);
```

## Как работает?

- PHP-классы формируют метаданные.
- Класс Space преобразует метаданные в HTML.

## Установка

```bash
git clone https://github.com/o-planet/lotis.git

```
