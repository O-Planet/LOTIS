# LOTIS — фреймворк для быстрой разработки WEB-приложений на PHP

## Что такое LOTIS? 
LOTIS (Low Time Script) — быстрый и компактный фреймворк для создания веб-приложений на PHP. 
Он фокусируется на производительности и легкости разработки, уменьшая количество ручного труда при создании интерфейсов и логики.
Вы строите приложение как при десктопной разработке, используя принцпы ООП без привязки в верстке.   

## Минимальный рабочий пример
```php
include_once 'newlotis/lotis.php';
$div = LTS::Div('main')->capt("Привет из LOTIS!");
LTS::Space()->build($div);
```

## Особенности
  - Zero-config: работает сразу после подключения, не требует дополнительной конфигурации.  
  - Единый язык программирования: все на PHP без ручной верстки.  
  - Работа как в десктопном приложении, в едином пространстве переменных без разделения на AJAX.
  - Использование бизнес-сущностей: таблиц, документов с табличными частями, стоков и коллекторов данных, многопользовательской синхранизации
  - Легко интегрируется с MySQL и jQuery.  

## Базовые концепции
  - **Объекты и классы**  
    Вся разработка строится вокруг классов, таких как `Div`, `Button`, `Form`, `DataTable`, которые образуют UI-интерфейсы и управляют данными.
    
  - **Метаданные, а не верстка**  
    Классы не генерируют непосредственный HTML, а лишь собирают метаданные, которые позже используются для формирования конечной страницы.
    
  - **Десктопность**  
    Приложение не делится на клиентскую и серверную части. Вся разработка ведется последовательно, позволяя концентрироваться на логике.
    
  - **Единое пространство имен**  
    Имена PHP-переменных, ссылающихся на объекты приложения, могут использоваться в коде клиентских скриптов для доступа к их свойствам и методам.
    
  - **События и глобальные переменные**  
    Обеспечивают взаимодействие между клиентами и серверами, поддерживая синхронизацию данных в течение всей сессии.
    
  - **Расширяемость и контроль**  
    В распоряжении разработчика хуки, сигналы, пользовательские методы.
    
  - **Сборка приложения**  
    Все элементы собираются в единое целое через вызов метода `Space::build()`, который превращает метаданные в реальный HTML.

## Начало работы

1. **Установка:**  
   Скачай репозиторий GitHub или добавь папку `src/newlotis` в свой проект.

2. **Точка входа:**  
   Подключи `lotis.php` и создай первый объект.

   ```php
   include_once 'newlotis/lotis.php';
   $div = LTS::Div('main')->capt('Привет из LOTIS!');
   ```

3. **Формирование интерфейса:**  
   Используй конструкторы классов для создания элементов и объединения их в композицию.

   ```php
   $loginform = LTS::Form()
       ->text('username', 'Логин')
       ->password('password', 'Пароль')
       ->button('login', 'Войти');
   $div->add($loginform);
   ```

4. **Управление стилями:**  
   Подключай css-файлы или управляй стилями непосредственно из PHP.

   ```php
   $div->css()->add('index.css');
   $loginform->css('border', 'solid 1px');
   ```

5. **События и обработка данных:**  
   Создавай события и работай с ними без AJAX.

   ```php
   $loginform->event('login', function($args) {
       return $args['username'] == 'Бред Пит' && $args['password'] == '123';
   });
   $loginform->event('login',
   <<<JS
       if(result) alert('Привет!'); else console.log('Иди на ...');
   JS
   );
   ```

6. **Гибкое управление логикой:**  
   Создавай собственные клиентские методы, внедряй хаки.

   ```php
   $loginform->method('mymethod',
   <<<JS
       alert('Все работает!');
   JS
       );
   $loginform->add('cheklogin', function ($args) {
        if($args['username'] == '' || $args['password'] == '') return false;
        return true;
   });
   ```
   
7. **Десктопность, единое пространство имен:**  
   Используй одни и те же названия переменных как в PHP-коде, так и в JS-скриптах, и не заморачивайся с ID объектов. 

   ```php
   $loginform->button('close', 'Закрыть')->click(
       <<<JS
           $(loginform).hide();
       JS
   );
   ```

8. **Сборка приложения:**  
   Точка сборки - класс Space. Каждый объект должен иметь родителя. Не важно, в какой момент ты его назначишь. Но Space - общий родитель для всех.

   ```php
   LTS::Space()->build($div);
   ```
